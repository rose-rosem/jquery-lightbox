Lightbox的效果类似于WinXP操作系统的注销/关机对话框，除去屏幕中心位置的对话框，其他的区域都以淡出的效果逐渐变为银灰色以增加对比度，此时除了对话框内的表单控件，没有其他区域可以获取。

Getting started How to use ?


1.Download the latest version as a zip file

2.Open up the zip file and take a peek at the barebones, working example that is included in the /examples folder.

3.Ready to set Lightbox up on your page? Start by including the Lightbox CSS and Javascript. You can grab both these files from the /dist folder.

  • Include the CSS at the top of your page in your <head>tag: 

    <link rel="stylesheet" href="css/style.css">
  • Include the Javascript at the bottom of your page before the closing <body>tag: 

    <script src="js/lightbox.js"></script>

4.Make sure jQuery, which is required by Lightbox, is also loaded.

  • If you already use jQuery on your page, make sure it is loaded before lightbox.js. jQuery 1.7 or greater is required.

5.Confirm that the four images loaded by lightbox.css are in the correct location. You can grab the images from the images folder.


Initialize with HTML.


  • You need to add class= "js-lightbox" or data-role= "Lightbox" on the img tag to make sure you call the function. 

    <img class="js-lightbox" data-role="lightbox" data-source="images/1-1.jpg" data-id="a0" data-caption="xxxxxxx">
  • If you have a group of related images that you would like to combine into a set, use the same data-lightbox attribute value for all of the images.
    For example: 
    <img class="js-lightbox" data-role="lightbox" data-source="images/1-1.jpg" data-group="group-1" data-id="b0" data-caption="xxxxxxx">
<img class="js-lightbox" data-role="lightbox" data-source="images/1-2.jpg" data-group="group-1" data-id="b1" data-caption="xxxxxxx">
<img class="js-lightbox" data-role="lightbox" data-source="images/1-3.jpg" data-group="group-1" data-id="b2" data-caption="xxxxxxx">
    class="js-lightbox"            表示这个元素要启用lightbox 
    
data-role="lightbox"           表示这个元素要启用lightbox 
    
data-source="images/1-1.jpg"   原图的地址         
    
data-group="group-1"           标识当前是否属于一个组别    
    
data-id="a0"                   图片的唯一标识             
    
data-caption="ajgjhj"          当前图片的描述文字

Options Change any of the default options!


If you want to change any of the default options, call the option method.


    <script>
        $(function(){
            var lightbox = new LightBox({

                speed:500,     //弹出速度
                winScales:0.8  //视口比例

            });
        });
    </script>
